# SmartJFB 专业体育比分系统

> 🏆 专业级体育比分管理系统 - 支持篮球、足球等多种运动

---

## ⚡ 30秒快速开始

### 1. 启动系统

**Windows用户**：
```
双击运行：启动.bat
```

**Linux/Mac用户**：
```bash
chmod +x start.sh
./start.sh
```

### 2. 打开页面

**方法1 - 快速访问**：
```
双击运行：快速访问.bat
```

**方法2 - 浏览器访问**：
```
http://localhost:3001/
```

### 3. 开始使用

1. 打开管理面板设置队伍
2. 开始比赛计时
3. 记录比赛事件
4. 查看实时统计

## 💡 使用技巧

### 推荐工作流
1. 双击 `启动.bat` 启动系统
2. 双击 `快速访问.bat` 打开页面
3. 管理面板（技术台）操作比赛
4. 记分牌投放到大屏幕
5. 比赛后导出数据

### 多屏幕设置
- **主记分牌** → 现场大屏幕
- **管理面板** → 技术台电脑
- **统计面板** → 辅助显示器
- **监控器** → 技术台内部

## ❓ 常见问题

### Q: 端口被占用怎么办？
A: 启动脚本会自动检测并尝试关闭占用端口的进程，或手动关闭其他服务。

### Q: 数据会自动保存吗？
A: 是的，所有操作实时保存到JSON文件。

### Q: 如何导出比赛数据？
A: 打开导出页面，选择内容和格式，点击导出即可。

### Q: 支持哪些运动？
A: 目前支持篮球和足球，架构支持扩展更多运动。

### Q: 可以自定义记分牌布局吗？
A: 可以，使用布局编辑器创建自定义布局。
