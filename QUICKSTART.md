# 🏀 体育比分工具 - 快速入门

## 5分钟快速开始

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动应用

**Windows:**
```cmd
start.bat
```

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

**或者直接使用:**
```bash
npm start
```

### 3️⃣ 访问应用

- **标准版**: http://localhost:3000
- **开发版**: http://localhost:3000/dev/index.html

---

## 🔑 激活码使用

### 获取机器码
1. 打开标准版应用
2. 点击右上角"激活"按钮
3. 复制显示的机器码

### 生成激活码

**Windows:**
```cmd
generate_activation.cmd standard <机器码> 365
```

**Linux/Mac:**
```bash
./generate_activation.sh standard <机器码> 365
```

**开发版（无需机器码，永久使用）:**
```bash
node generate_activation.js dev
```

### 激活应用
1. 在应用中点击"激活"
2. 粘贴生成的激活码
3. 点击"确认"完成激活

## 🎯 核心功能

### 添加队员
1. 选择"主队"或"客队"
2. 输入队员姓名和号码
3. 点击"添加"
4. 或点击"导入"批量导入

### 记录得分/射门
1. 点击要记录的队员
2. 点击对应的动作按钮
3. 系统自动更新比分和统计

### 判罚记录
1. 选择犯规队员
2. 点击犯规类型按钮
3. 系统自动记录犯规次数

### 导出数据
1. 点击"导出Excel"
2. CSV文件自动下载
3. 包含完整的比赛记录和统计

## 📊 比赛流程示例

### 篮球比赛
```
1. 添加主队队员
2. 添加客队队员
3. 点击"开始"启动计时器
4. 选择队员，记录得分（3分/2分/罚球）
5. 记录犯规
6. 比赛结束后导出数据
```

### 足球比赛
```
1. 切换到"足球"模式
2. 添加队员
3. 记录射门/点球/任意球
4. 记录红黄牌
5. 导出比赛数据
```

## 🆘 常见问题

### Q: 激活码无效？
A: 检查机器码是否正确，确保激活码为此设备生成

### Q: 激活码过期？
A: 重新生成新的激活码

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地，清除浏览器数据会丢失

### Q: 如何备份？
A: 定期导出Excel数据作为备份

### Q: 开发版和标准版区别？
A: 开发版无需激活、永久有效、功能完全相同

## 💡 使用技巧

1. **先添加队员**：比赛开始前添加所有队员
2. **及时记录**：每个动作及时记录，避免遗漏
3. **定期导出**：重要比赛及时导出数据备份
4. **主题选择**：夜间比赛使用深色主题更护眼
5. **批量导入**：队员较多时使用批量导入功能
