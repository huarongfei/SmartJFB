# 管理面板使用指南

## 🚀 快速开始

### 1. 启动系统
```bash
# Windows
双击：启动.bat

# Linux/Mac
./start.sh
```

### 2. 打开管理面板
```
http://localhost:3001/admin.html
```

### 3. 检查连接状态
- 页面右上角应显示"已连接"（绿色圆点）
- 如果显示"未连接"（红色圆点），请检查服务器是否启动

---

## 👥 球员管理

### 添加球员

#### 主队
1. 在"主队"区域，点击"添加球员"按钮
2. 输入球衣号码（如：23）
3. 输入球员姓名（如：张三）
4. 点击"确认添加"

#### 客队
1. 在"客队"区域，点击"添加球员"按钮
2. 输入球衣号码（如：7）
3. 输入球员姓名（如：李四）
4. 点击"确认添加"

### 删除球员
1. 找到要删除的球员卡片
2. 点击"删除"按钮
3. 确认删除操作

### 查看球员数据
- 球员卡片实时显示：
  - 球衣号码（圆形徽章）
  - 球员姓名
  - 得分
  - 犯规数

## 📝 记分操作

### 投篮得分记录

#### 步骤1：选择队伍
在"记分操作"区域，选择：
- 主队
- 客队

#### 步骤2：选择球员
选择投篮的球员（下拉列表中显示）
- 格式：`#23 张三`

#### 步骤3：选择分值
- **1分** - 罚球命中
- **2分** - 中投/上篮
- **3分** - 三分球

系统自动记录：
- ✅ 队伍
- ✅ 球员姓名
- ✅ 球衣号码
- ✅ 得分
- ✅ 时间
- ✅ 节数

### 记录犯规
1. 选择队伍
2. 选择犯规球员
3. 点击"犯规"按钮

### 记录暂停
1. 选择请求暂停的队伍
2. 点击"暂停"按钮

### 修正比分错误
如果多加了分数：
1. 选择队伍
2. 点击"-1分（修正）"按钮

## ⏱️ 计时控制

### 比赛时钟

#### 基本操作
- **开始** - 开始计时
- **暂停** - 暂停计时
- **继续** - 继续计时
- **重置** - 重置为初始时间

#### 快速调整
- **+1分钟** - 增加比赛时间（补时）
- **-1分钟** - 减少比赛时间
- **设置时间** - 自定义时间

#### 预设时间
点击"设置时间"后，可快速选择：
- 12分钟（标准NBA）
- 10分钟（FIBA）
- 8分钟
- 5分钟

#### 节次管理
- 点击"本节结束"进入下一节
- 切换节数：第1节 → 第2节 → 第3节 → 第4节 → 加时赛

### 进攻时钟

#### 基本操作
- **开始** - 开始倒计时
- **暂停** - 暂停倒计时
- **重置** - 重置为初始时间

#### 快速设置
- **24秒** - 标准进攻时间
- **14秒** - 前场篮板后进攻时间

## 🏆 比分显示

### 实时比分
- 主队分数（左侧）
- 客队分数（右侧）
- 超大字体显示，易于查看
- 渐变色彩，视觉效果好

### 更新队名
1. 在"比赛信息"区域，找到队伍名称输入框
2. 输入新队名
3. 点击"更新队名"按钮

## 📊 事件记录

### 查看事件
在"事件记录"区域，可以看到所有比赛事件：
- 🏀 得分
- ⚠️ 犯规
- ⏸️ 暂停
- 📋 节次

每条事件显示：
- 队伍名称
- 球员信息（如果相关）
- 事件描述
- 发生时间（节数 + 比赛时间）

### 删除事件
1. 找到要删除的事件
2. 点击"删除"按钮
3. 系统自动回退该事件的统计数据

### 清空所有事件
1. 点击"清空所有事件"按钮
2. 确认清空操作
3. 所有事件和统计数据将被重置

## 💡 使用技巧

### 快速操作
1. **批量添加球员** - 一次性添加所有上场球员
2. **预设时间** - 使用预设时间快速设置
3. **队伍切换** - 记分后保持队伍选择，减少点击

### 错误修正
1. **多加分数** - 立即使用"-1分"修正
2. **选错球员** - 删除事件，重新记录
3. **时间错误** - 使用"+/-1分钟"快速调整

### 数据保存
- 所有数据自动保存到服务器内存
- 刷新页面不会丢失数据
- 停止服务器会清空数据

## ⚠️ 注意事项

### 连接问题
- 如果显示"未连接"：

2. 设置队名和比赛时间
3. 开始比赛
4. 依次记录事件

## 📞 常见问题

### Q: 无法连接到服务器？
A: 检查服务器是否启动，运行 `启动.bat` 重启服务器

### Q: 球员添加失败？
A: 确保球衣号码和姓名都已填写

### Q: 如何修改已记录的比分？
A: 使用"-1分（修正）"按钮或删除对应事件

### Q: 进攻时钟不工作？
A: 点击"重置"后再点击"开始"

### Q: 如何恢复误删的事件？
A: 无法恢复，请谨慎删除操作
